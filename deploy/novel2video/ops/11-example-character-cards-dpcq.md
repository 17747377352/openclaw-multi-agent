# 示例角色卡 - 《斗破苍穹》前期篇章

目标：给前期高频角色建立稳定生产标准。

推荐优先级：

1. 萧炎
2. 药老
3. 纳兰嫣然
4. 薰儿

---

## 1. 萧炎

### 基础身份

- 角色名：萧炎
- 性别：男
- 年龄段：少年
- 身高感：中等偏高
- 体型：瘦削但结实
- 阵营：萧家
- 角色定位：主角

### 外形硬锁定项

- 脸型：偏瘦的少年脸，下颌线清晰但不成熟
- 眼型：细长、略上挑，目光锐利
- 瞳色：深棕近黑
- 鼻型：直鼻、少年感明显
- 唇形：薄唇，常微抿
- 发型大轮廓：黑色高束发，后部扎起，额前有碎发
- 发色：纯黑偏冷调
- 刘海特征：额前碎发有层次，不厚重遮眼
- 眉形：略剑眉，年轻感强
- 身材比例：肩不宽，四肢利落，少年武者感
- 皮肤基调：偏白但不病态

### 服装硬锁定项

#### 常服

- 上装：深灰或黑色劲装上衣
- 下装：深色窄腿裤
- 腰部结构：暗红或深色腰封
- 鞋靴：深色短靴
- 主配色：黑、深灰、暗红少量点缀
- 面料风格：轻便布料，适合行动
- 标志细节：发带、简洁护腕

#### 战斗状态

- 不换一整套新衣
- 只允许衣摆更凌厉、护腕更明显、局部特效加强

### 标志物

- 黑色发带
- 母亲留下的戒指/玉佩类重要物件
- 火属性斗气色彩

### 性格可视化

- 常态表情：克制、倔强、不轻易笑
- 被羞辱时：眼神更冷，嘴角收紧
- 愤怒时：眉压低、拳头握紧、肩膀紧绷
- 独处时：低头沉思、目光发沉

### 角色生成锚点

`teenage chinese fantasy male, same black tied hair, same dark martial outfit, slim but athletic, sharp eyes, restrained anger, consistent character design`

---

## 2. 药老

### 基础身份

- 角色名：药老
- 性别：男
- 年龄段：老年
- 身高感：高挑
- 体型：清瘦但挺拔
- 阵营：萧炎之师
- 角色定位：导师 / 强者 / 引路人

### 外形硬锁定项

- 脸型：瘦长老者脸，骨相清晰
- 眼型：不大，但非常有神
- 瞳色：灰褐或偏深
- 发型：长白发，后梳，有层次
- 发色：白色带冷灰
- 胡须：长白须，整洁不凌乱
- 眉形：长眉，略垂
- 气质：慈祥但威严，不软弱

### 服装硬锁定项

- 白色或浅灰长袍
- 宽袖
- 轻微金纹或低调纹样
- 灵魂体状态下衣袍边缘可有淡光

### 标志物

- 白袍
- 白发白须
- 灵魂体淡青白光晕
- 炼药师/强者气场

### 性格可视化

- 常态：淡定、从容、睿智
- 训诫时：眼神更沉，眉略压
- 认可萧炎时：表情放松但不夸张

### 角色生成锚点

`elder chinese fantasy master, white long hair, white beard, same white robe, calm wise expression, subtle spiritual glow, strong presence, consistent character design`

---

## 3. 纳兰嫣然

### 基础身份

- 角色名：纳兰嫣然
- 性别：女
- 年龄段：少女 / 青年之间
- 身高感：高挑
- 体型：修长匀称
- 阵营：云岚宗
- 角色定位：高位压迫者 / 冲突推动者

### 外形硬锁定项

- 脸型：偏长鹅蛋脸，轮廓干净
- 眼型：细长偏冷，不圆润可爱
- 瞳色：深灰或冷棕
- 发型：黑发或深色长发，束发利落
- 发色：深黑偏冷
- 眉形：平直中带锐感
- 气质：高傲、冷静、受过严格训练

### 服装硬锁定项

- 白色劲装为主
- 腰线清晰
- 背负长剑
- 配色：白、银、淡青
- 面料：干净利落，不飘逸仙裙化

### 标志物

- 白色劲装
- 长剑
- 高位者站姿

### 性格可视化

- 常态：下巴微抬，目光稳定
- 对峙时：不夸张愤怒，更多是坚定和疏离
- 面对萧家时：压迫感来自姿态，不来自狰狞表情

### 角色生成锚点

`young chinese fantasy female swordswoman, same white martial outfit, same long sword, elegant and proud, cold expression, clean posture, consistent character design`

---

## 4. 薰儿

### 基础身份

- 角色名：薰儿
- 性别：女
- 年龄段：少女
- 身高感：中等偏高
- 体型：纤细
- 阵营：古族
- 角色定位：温柔支撑 / 女主气质角色

### 外形硬锁定项

- 脸型：柔和小巧鹅蛋脸
- 眼型：柔长型，不夸张上挑
- 瞳色：金棕或偏浅暖色
- 发型：黑色长发，顺直如瀑
- 发色：黑色偏暖
- 气质：温柔、高贵、带距离感

### 服装硬锁定项

- 淡紫或浅青紫衣裙
- 裙摆柔和但不复杂
- 配饰低调精致
- 面料有轻盈贵气感

### 标志物

- 淡紫色长裙
- 温柔眼神
- 安静但不弱的气质

### 性格可视化

- 常态：平静柔和
- 关心萧炎时：目光变柔，嘴角轻微变化
- 与外人相处时：保持分寸感和距离感

### 角色生成锚点

`young chinese fantasy noble girl, same long black hair, same pale violet dress, elegant and gentle, distant but warm gaze, refined temperament, consistent character design`

## 5. 角色生产建议

实际产图顺序：

1. 先出萧炎标准卡
2. 再出药老标准卡
3. 再出纳兰嫣然标准卡
4. 最后补薰儿

原因：

- 前期剧情高频冲突主要围绕萧炎、药老、纳兰嫣然
- 先把高频角色控住，一致性收益最大
