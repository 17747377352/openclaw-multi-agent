## 💰 优化建议

### 1. 批量生成折扣
- 火山引擎企业账户可申请 **9 折** 优惠
- 月消费 >¥5000 可谈 **8 折** 协议价

### 2. 缓存策略
- 相同人物描述 → 复用已生成的人物照
- 相似场景 → 复用分镜提示词模板

### 3. 失败重试控制
- 单任务最多重试 3 次
- 连续失败 3 次 → 转人工审核

### 4. 视频时长优化
- 默认 5 秒，可根据场景调整为 3-8 秒
- 时长减半 = 成本减半（部分场景适用）

### 5. OSS 冷存储
- 完成项目转 **低频访问** (¥0.08/GB/月)
- 归档项目转 **归档存储** (¥0.033/GB/月)
