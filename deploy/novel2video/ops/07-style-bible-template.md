# 2D 动态漫画 - 风格圣经模板

适用目标：小说改编短视频、动态漫画、2.5D 分镜视频。

使用方式：

1. 每个项目先填这一份，后面所有人物卡、分镜提示词、配音、剪辑都必须服从这里。
2. 这里定义的是“项目级不变量”，不是单个镜头的创意发挥。
3. 没写进来的内容，默认不允许模型自由发挥。

## 2. 视觉路线总定义

这一节只能选一个主路线，不要混搭。

### 主视觉路线

- 画风关键词：国风 2D 动漫 / 少年热血 / 轻写实插画 / 日系赛璐璐 / 水墨幻想 / 新海诚系光感 / 游戏立绘风
- 线条风格：硬朗清晰 / 柔和细线 / 粗线条漫画感
- 上色风格：赛璐璐平涂 / 半厚涂 / 厚涂插画
- 细节密度：低 / 中 / 高
- 材质倾向：布料清晰、金属明确、皮肤干净，不追求照片级写实
- 阴影规则：单主光源 / 双层阴影 / 柔和环境光

### 推荐默认路线

- 画风关键词：`国风少年热血 2D 动漫`
- 线条风格：`清晰稳定的中细线`
- 上色风格：`赛璐璐为主，少量半厚涂增强重点镜头`
- 细节密度：`中`
- 阴影规则：`单主光源 + 明确边缘阴影`

### 明确禁止

- 禁止写实照片感皮肤
- 禁止不同镜头切换不同画风
- 禁止某一镜头突然变 3D 游戏渲染感
- 禁止人物五官比例大幅变化
- 禁止背景精度远高于人物或远低于人物

## 5. 镜头语言圣经

### 镜头占比建议

- 远景：20%
- 中景：40%
- 近景：30%
- 特写：10%

### 常用镜头动作

- 推镜
- 拉镜
- 横移
- 轻微跟随
- 视差分层移动

### 不推荐动作

- 长时间复杂旋转镜头
- 高频快速乱晃
- 没必要的无人机感大运镜
- 连续多段高难动作镜头

### 剪辑节奏建议

- 对话镜头：3-6 秒
- 情绪停顿镜头：2-4 秒
- 动作镜头：1.5-3 秒
- 大场面建立镜头：2-3 秒

结论：

- 你做的是“可量产叙事”，不是炫技动画 demo。
- 镜头稳定比花哨更重要。

## 6. 人物一致性总规则

### 所有人物必须固定的项

- 脸型
- 五官比例
- 发型轮廓
- 发色
- 代表服装
- 身材比例
- 标志物

### 可变化的项

- 表情
- 站姿
- 手势
- 衣摆动态
- 轻微受伤/脏污
- 光照角度

### 禁止漂移项

- 年龄感忽老忽少
- 发型层次变化过大
- 衣服结构突然变款式
- 同一角色眼睛形状变化太大
- 同一角色瘦胖变化明显

硬规则：

- 每个核心角色必须先出一张 `角色标准卡`
- 再出 `正脸 / 侧脸 / 半身 / 全身 / 表情组`
- 分镜和视频都以角色卡为准，不允许只靠小说描述自由生成

## 7. 背景与场景规则

### 主要场景母版

至少定义这些固定场景：

- 主角房间 / 洞府 / 居所
- 宗门/学院主广场
- 训练场
- 山林/野外
- 城市街道
- 冲突场景主空间

### 每个场景要固定

- 建筑结构关键词
- 主色调
- 光线方向
- 常驻摆件/符号
- 镜头常用角度

原则：

- 同一场景不要每次都重新想象
- 应先有“场景母版”，再从母版衍生镜头

## 8. 动效预算规则

### 建议可做的动效

- 镜头推拉
- 分层视差
- 风吹衣摆/头发
- 眼睛眨动
- 嘴型开合
- 烟雾、火焰、能量流、落叶、光斑

### 不建议默认做的动效

- 连续打斗肢体动画
- 复杂多人群像同步动作
- 精细手部表演
- 连续走路循环的大段镜头

生产原则：

- 用镜头语言和特效补运动感
- 不用高风险角色运动硬拼“真动画”

## 11. 成片验收清单

每集/每批镜头验收时，必须过这一遍：

- 角色脸是否统一
- 角色服装是否统一
- 发型是否统一
- 场景是否属于同一世界观
- 光影和上色是否统一
- 镜头时长是否适合短视频节奏
- 字幕是否统一
- 配音和画面气质是否统一
- 是否存在明显 AI 漂移镜头
- 是否有“单镜头特别丑”拖累整条成片

验收标准：

- 宁可删掉一个丑镜头，也不要勉强保留

---

## 12. 推荐制作流程

1. 填写风格圣经
2. 输出核心角色卡
3. 输出主要场景母版
4. 生成分镜草稿
5. 审核提示词
6. 先出首帧图
7. 首帧通过后再做动态视频
8. 最后统一配音、字幕、BGM、剪辑

一句话原则：

- 先统一“资产”，再批量“出镜头”，最后才是“动起来”。
