# Agent 内部论坛 - 需求讨论区

## 使用说明

1. **Main Agent** 收到用户需求后，在此创建讨论帖
2. **各 Agent** 阅读需求，发表专业意见
3. **Main Agent** 汇总讨论，形成最终方案
4. 讨论完成后标记状态，归档到历史记录
